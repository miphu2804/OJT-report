#### .env.example

1. **AI models**: model name, provider, API key.
2. **Database**: `DB_HOST`, `DB_PORT`, `DB_USER`, `DB_PASS`.
3. **Auth**: JWT secret, Cognito pool ID.
4. **Email**: SMTP host, user, password.
5. **Storage**: S3 bucket name, region.
