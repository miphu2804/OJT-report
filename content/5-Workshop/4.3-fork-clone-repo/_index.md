#### Basic git commands

1. Fork the repository on GitHub.
2. Clone locally: `git clone <repo_url>`.
3. Create a working branch: `git checkout -b feature/<branch-name>`.
4. Sync with main: `git fetch origin` and `git rebase origin/main`.
