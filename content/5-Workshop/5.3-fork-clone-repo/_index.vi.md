#### Các lệnh git cơ bản

1. Fork repository trên GitHub.
2. Clone về máy local: `git clone <repo_url>`.
3. Tạo nhánh làm việc: `git checkout -b feature/<ten-nhanh>`.
4. Cập nhật từ main: `git fetch origin` và `git rebase origin/main`.
